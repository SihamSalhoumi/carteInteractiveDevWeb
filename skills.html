Savoir-être

Coopération

Un développeur travaille principalement en collaboration sur divers projets. L'esprit d'équipe est une composante de mon métier actuel que j'apprécie particulièrement.

Esprit logique

Mon sens de la logique sera utile dans ce nouveau métier où la recherche de solutions est primordiale.

Curiosité

Ma curiosité me permet de réaliser naturellement une veille technologique dans un secteur en évolution constante.

Gestion du stress

J'ai acquis cette compétence grâce à mes 10 ans d'expérience dans les assurances. Une bonne gestion du stress est indispensable dans le domaine informatique pour faire face aux imprévus.

Autodidaxie

J'apprécie m'instruire au quotidien et me perfectionner en permanence. Ceci est très utile dans un secteur qui nécessite une autonomie rapide.

Compétences techniques

HTML / CSS

Ces deux langages informatiques sont la base de tout site web. Je m'entraîne régulièrement à maîtriser HTML et CSS depuis plus d'un an par le biais de e-learning.

Algorithme

Un algorithme est une procédure par étapes, c'est un ensemble de règles à suivre pour accomplir une tâche ou résoudre un problème. Maîtriser la logique algorithmique permet d'assimiler plus facilement n'importe quel langage. C'est pourquoi je m'entraîne souvent dans ce domaine.

Git / Github

Git est un logiciel de versionning permettant de conserver l'historique des modifications d'un site. Github est une plateforme de projet collaborative. Ces deux outils sont interconnectés et très populaires dans l'informatique. Je les utilise également pour mes projets.

Vision utilisateur et tests d'outils

Dans mon emploi actuel, j'ai participé à plusieurs projets liés à de nouveaux logiciels pour les tester en amont.

J'apprécie identifier les améliorations à apporter dans le but de satisfaire le client. Cette vision "utilisateur" est très utile dans le développement de sites internet.

Compétence additionnelle : La photographie

J'aime photographier des paysages ou des objets du quotidien. La qualité des images joue un rôle primordial dans le design d'un site web. Cette passion est à mes yeux un atout non négligable. Il me sera en effet possible d'associer cette activité avec ma future profession.
