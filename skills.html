Compétences techniques

HTML / CSS

C’est deux langages sont la base de tous site web. Je m'entraîne à maîtriser HTML et CSS depuis plus d'un an par le biais de e-learning.

Algorithme

L'Algorithme est une suite d'instruction reprensentant la solution à un problème de programmation. Maîtriser la logique algorithmique permet d'assimiler plus facilement n'importe quel language. C'est pourquoi je m'entraine souvent dans ce domaine.

Git / Github

Git est un logiciel de versionning permettant de conserver l'historique des modifications. Github est une plateforme de projet collaborative. Ces deux outils sont interconnectés et très populaires dans l'informatique. Je maîtrise et utilise également ces deux outils pour mes projets.

Vision utilisateur et test d'outils

Dans mon emploi actuel, j'ai aimé participer à plusieurs projets liés à de nouveaux logiciels pour les tester en amont.

J'aime identifier les améliorations à apporter dans le but de satisfaire le client final. Cette vision "utilisateur" est très utile dans le développement de site internet.

Autre atout : La photographie

Je suis passionnée de photographie, j'aime prendre des photos de paysages ou d'objet du quotidien. La qualité des images joue un rôle primordiale dans la réussite d'un site web. Cette passion est pour moi un plus dans mon futur métier. Il me sera en effet possible d'associer cette activité avec mes futures compétences dans le développement web pour crée des sites web attractifs comportant des photos personnalisés.
